# פרויקט הנבטאים - דרך הקטורת

## 📋 תיאור הפרויקט

פרויקט היסטוריה משולב המחקה את תרבות הנבטאים העתיקה וערייהם בנגב. הפרויקט כולל שלוש חלקים עיקריים, כולם בעברית וברמה גבוהה של עיצוב.

---

## 📦 הקבצים המסופקים

### 1. **1pager-nabataeans.html** - נייר אחד מקיף
- **תוכן**: הסבר מקיף על הנבטאים, מקורם, תרבותם, סחרם, ארכיטקטורה
- **קהל יעד**: תלמיד כיתה ו' עד בוגרים
- **עיצוב**: ארכיאולוגי עתיק עם קוורץ מודרני, צבעי זהב וחול
- **יכולת**: הדפסה ל-PDF, צפייה דיגיטלית
- **אורך**: דף אחד מלא עם מידע עשיר

### 2. **postcard-avdat.html** - גלויה מעבדת
- **תוכן**: גלויה אישית כאילו מטיול בעבדת
- **עיצוב**: עיצוב גלויה קלסי עם בול וחותמת
- **שפה**: עברית אליגנטית וטבעית
- **שימוש**: הדפסה, שליחה, תצוגה
- **אורך**: טקסט קצר ותיאורי

### 3. **interactive-website.html** - אתר אינטראקטיבי
- **תוכן**:
  - 6 עובדות מרתקות (2-3 כלליות + 1 לכל עיר)
  - ניווט אינטראקטיבי בין 3 הערים
  - מידע מפורט על כל עיר
- **עיצוב**: עתיק אך מודרני, אנימציות, הפעלות עכברי
- **תכונות**:
  - כפתורים עם משוב
  - אנימציות של כניסה
  - ממשק תגובתי (ניידות וקריאות)

## 🎨 עיצוב עקבי בכל שלוש הקבצים

**צבעים ארכיאולוגיים**:
- זהב: #f4a460
- זהב כהה: #c4a062
- חול: #e8d7c3
- רקע: שחור/חום כהה

**טיפוגרפיה**:
- כותרות: Playfair Display (אלגנטי ודרמטי)
- גוף: Crimson Text (קראות וקלאסי)
- ניווט: Montserrat (מודרני וברור)

**אנימציות והשפעות**:
- כניסה חלקה של אלמנטים
- הובלה וסקרול ממושך
- אפקטים של זוהר וחסכון של טקסט

## 🚀 איך להשתמש

### צפייה
1. פתח כל קובץ HTML בדפדפן
2. יהנה מחוויה אינטראקטיבית

### הדפסה
1. בחר "הדפסה" מתפריט הדפדפן (Ctrl+P / Cmd+P)
2. בחר "שמור ל-PDF" או "הדפסה לנייר"
3. התאם הגדרות לפי הצורך

### שיתוף
- שלח כקבצים HTML ישירות
- העלה לאתר אינטרנט
- הצג למורה או לחברים
